
## Getting Started

1. Go to this website: https://pytorch.org/get-started/locally/.
2. Select your OS, Libtorch, C++/Java, and choose your build platform.
3. Unzip the files and put the libtorch folder into external/libtorch inside the root of this repo.
4.     mkdir build && cd build
5.     cmake ..
6.     make
7.     ./main
